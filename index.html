URL shortener microservice

Pass URL address as pathname, get shortened URL as response. Visiting generated short URL will redirect you to the original address.

Example usage:

https://secret-meadow-30442.herokuapp.com/shorten/https://www.google.com

Example output:

{"original_url":"https://www.google.com","short_url":"https://secret-meadow-30442.herokuapp.com/eiua"}

Example usage of short URL:

Visiting https://secret-meadow-30442.herokuapp.com/uZKC will redirect you to https://www.google.com
